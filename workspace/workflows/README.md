# Workflows de n8n

Este directorio contiene los flujos de trabajo exportados de n8n.

## Cómo exportar un workflow desde n8n

1. Abre tu workflow en n8n
2. Haz clic en el menú (tres puntos)
3. Selecciona "Download"
4. Guarda el archivo JSON en este directorio

## Cómo importar un workflow a n8n

1. En n8n, haz clic en "Add workflow"
2. Selecciona "Import from File"
3. Selecciona el archivo JSON de este directorio
