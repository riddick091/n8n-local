# Credenciales

**⚠️ IMPORTANTE: NO incluyas credenciales reales en este directorio**

Este directorio es para almacenar plantillas de credenciales o documentación sobre qué credenciales se necesitan para los workflows.

## Ejemplo de documentación de credenciales

Puedes crear archivos que describan las credenciales necesarias:

```
workflow-ejemplo.md
- API Key de servicio X
- OAuth para servicio Y
- Webhook URL
```

Nunca incluyas:
- Contraseñas reales
- API keys reales
- Tokens de acceso
- Datos sensibles
